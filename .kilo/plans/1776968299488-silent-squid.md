## 🎯 AFTER CREATING PYTHONANYWHERE ACCOUNT - SIMPLE STEPS

**এখন কি করবেন:**

### **স্টেপ ১: প্রোজেক্ট কোড পায়thonAnywhere এ আনা**

**বিকল্প A (GitHub - সহজ):**
```bash
# PythonAnywhere এ Bash console খুলুন
# GitHub থেকে clone করুন:
git clone https://github.com/YOUR_USERNAME/Rent_a_Car.git
cd Rent_a_Car
```

**বিকল্প B (Manual Upload):**
1. PythonAnywhere Dashboard → **Files** tab
2. `~/` এ গিয়ে **Upload** বাটন ব্যবহার করে প্রোজেক্টের ZIP আপলোড করুন
3. Extract করুন → `Rent_a_Car` folder他向 holo

---

### **স্টেপ ২: Virtualenv তৈরি (Bash console)**

```bash
cd ~/Rent_a_Car
python3 -m venv venv
# Verify: ls -la দেখুন যে `venv/` folder ase
```

---

### **স্টেপ ৩: Dependencies ইনস্টল**

```bash
source venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
# যদি requirements.txt না থাকে:
# pip install django pillow xhtml2pdf
```

---

### **স্টেপ ৪: Environment Variables সেট করুন**

PythonAnywhere **Web** tab → **Code** → **Environment variables**:

| KEY | VALUE |
|-----|-------|
| `SECRET_KEY` | `python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"` থেকে পাওয়া |
| `DEBUG` | `False` |
| `ALLOWED_HOSTS` | `yourusername.pythonanywhere.com` |
| `EMAIL_HOST_USER` | আপনার Gmail |
| `EMAIL_HOST_PASSWORD` | Gmail App Password |

---

### **স্টেপ ৫: WSGI File কনফিগার**

1. **Web** tab → **WSGI configuration file** link ক্লিক
2. পুরো ফাইল নিয়ে নillos এ রিপ্লেস করুন:

```python
import os
import sys

path = '/home/yourusername/Rent_a_Car'
if path not in sys.path:
    sys.path.append(path)

os.environ['DJANGO_SETTINGS_MODULE'] = 'rent_a_car.settings'

activate_this = '/home/yourusername/Rent_a_Car/venv/bin/activate_this.py'
with open(activate_this) as f:
    exec(f.read(), {'__file__': activate_this})

from django.core.wsgi import get_wsgi_application
application = get_wsgi_application()
```

**yourusername** → আপনার PythonAnywhere username দিয়ে 바দলান

### **স্টেপ ৬: Static & Media Mapping (Web tab)**

**Web** tab → **Static files** section:

| URL | Path |
|-----|------|
| `/static/` | `/home/username/Rent_a_Car/staticfiles/` |
| `/media/` | `/home/username/Rent_a_Car/media/` |

---

### **স্টেপ ৭: Django কমান্ড রান (Bash console)**

```bash
cd ~/Rent_a_Car
source venv/bin/activate

python manage.py migrate
python manage.py collectstatic --noinput
python manage.py createsuperuser  # এবং Bangladesher စစျင်
```

---

### **স্টেপ ৮: Reload ও Test**

1. **Web** tab → **Reload** বাটন click
2. Browser খুলে যান: `https://yourusername.pythonanywhere.com`
3. Admin চেক: `https://yourusername.pythonanywhere.com/admin/`

**Problem হলে:** **Web** tab → **Error log** দেখুন

## 📋 TRY THESE COMMANDS SEQUENTIALLY

```bash
# 1. Clone (GitHub থেকে)
git clone https://github.com/YOUR_USERNAME/Rent_a_Car.git
cd Rent_a_Car

# 2. Virtualenv
python3 -m venv venv

# 3. Activate
source venv/bin/activate

# 4. Install
pip install django pillow xhtml2pdf

# 5. SECRET_KEY generate
python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"

# 6. Migrate
python manage.py migrate

# 7. Collect static
python manage.py collectstatic --noinput

# 8. Create superuser
python manage.py createsuperuser

# 9.आप Web tab এ goto → Reload
```
